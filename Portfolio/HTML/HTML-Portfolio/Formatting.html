HTML Formatting

<b>: Makes text bold but does not add strong semantic emphasis like <strong>.

<i>: Italicizes text, often used for emphasis or foreign words.

<u>: Underlines text, typically for emphasis or links.

<strong>: Adds bold styling with a strong emphasis, meaningful for screen readers.

<em>: italicizes text as well, but it adds a layer of emphasis that can change the tone of the sentence, signaling emphasis to screen readers.

<mark>: Highlights text with a background color to draw attention.

<small>: Displays text in a smaller font size, often used for side notes or fine print.

<del>: Shows text as deleted, typically with a strikethrough.

<ins>: underlines text to show it was newly added, which can be helpful for tracking changes.

<sub>: Displays text as subscript, useful in chemical formulas.

<sup>: Raises text slightly above the baseline, typically used for exponents or footnotes.
