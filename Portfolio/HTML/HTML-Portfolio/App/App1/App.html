What is HTML?

HTML (HyperText Markup Language) is a markup language used to create the structure of web pages. HTML consists of various tags to organize the content of a web page and specify how it should be displayed by browsers. HTML5 is the latest version, offering more interactivity and media support.

HTML Tags

HTML allows content creation using various tags. For example:

  • <h1>...</h1> - The heading tag is used to define page headings.
  • <p>...</p> - The paragraph tag organizes text into paragraphs.
  • <a>...</a> - The anchor tag provides links to another page or resource.
  • <img> - The image tag is used to add images to the page.

Structure of HTML

A typical HTML document consists of the following sections:

  • <!DOCTYPE html> - Declares that the document is HTML5.
  • <html> - The root element of the HTML document.
  • <head> - Contains meta information, styles, and the title.
  • <body> - Contains the visible content of the page.
