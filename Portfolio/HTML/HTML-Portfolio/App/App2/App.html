HTML Guide

This page is designed to help you learn the basics of HTML and the most commonly used tags. HTML is a markup language that forms the building blocks of web pages and allows you to structure content.

  • Basic Tags
  • Forms
  • Lists

Basic HTML Tags

  1. <h1> - <h6>: Header tags are used to divide page content into headings.
  2. <p>: The paragraph tag is used to create blocks of text.
  3. <a>: The anchor tag is used to create links to another page or URL.

Forms

HTML forms are used to collect data from the user. Below are some common form elements:

  • <input>: A form element used for user data input.
  • <textarea>: Used for multi-line text input.
  • <button>: A button used to submit the form or perform an action.

HTML Lists

There are two types of lists in HTML:

  • Unordered List (<ul>): Used to create bulleted lists.
  • Ordered List (<ol>): Used to create numbered lists.
